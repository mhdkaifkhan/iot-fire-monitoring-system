# 🔥 IoT Fire Monitoring System

A real-time IoT Fire Detection System built using ESP32, MQTT, Node.js and MongoDB.

---

## 🌐 Live Dashboard

https://iot-fire-monitoring-system.onrender.com/index.html

---

## 📡 System Architecture

ESP32 Sensors  
↓  
MQTT (HiveMQ Cloud)  
↓  
Node.js Backend (Render Cloud)  
↓  
MongoDB Atlas  
↓  
Live Web Dashboard

---

## 🚀 Features

- Real-time temperature monitoring
- Smoke detection
- Fire status detection (SAFE / WARNING / FIRE)
- Email alert when fire detected
- Live dashboard with charts
- Data history storage
- Cloud deployment
